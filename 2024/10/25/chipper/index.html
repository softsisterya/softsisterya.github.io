三、配置教程

小火箭配置教程

  1. 开启小火箭➡️配置，点击右上角加号，将链接粘贴并点击下载
  1. 下载成功后点击脚本栏右侧的ℹ️，点击HTTPS解密➡️凭证➡️生成新的CA凭证➡️安装凭证
  2. 安装成功后打开打开设置➡️一般/通用➡️VPN与装置管理，安装已下载的描述档
  3. 回到一般➡️关于本机➡️凭证信任设定将凭证信任按钮打开
  4. 回到小火箭开启HTTPS解密并保存设置
  5. 回到配置➡️模块➡️点击右上角＋号，将链接粘贴并点击下载

7.点击模块勾选➡️回到配置➡️点击脚本，使用配置
8.配置结束，可打开chipper点击验证人脸，会弹出邮箱验证

四、绑定手机号和信息更改

绑定手机号

  • 请确保验证完整再绑定手机号，若出现×或未完成验证，请勿绑定手机号！！！！
  • 未验证完整示例

  • 验证完整示例

  • 点击Phone Number➤选择国家➤输入号码➤点击Receive code another way（使用验证码绑定）➤选择号码Receive code via SMS➤输入验证码➤绑定成功
  • 注意：尼日利亚号码（+234）输入时需在号码前加个0

信息更改

1. 更改邮箱信息

  • 不要去换绑邮箱或添加邮箱！！！！
  • 更改好邮箱密码，像啥两步验证，辅助邮箱，辅助手机号等全部更改好
  • 踢掉登录邮箱的其他设备

2. 更改chipper信息

  • 更改pin码
    home（主页面）➤点击右上角头像➤Settings➤Change Chipper PIN➤输入原密码➤输入新密码➤重复新密码
  • 移出其他设备
    home（主页面）➤点击右上角头像➤Settings➤Manage Devices➤点击设备名称旁的三个点➤选择移除

五、入金、开卡、转奶

交易所入金（trc20演示）

  • home（主页面）➤点击美国图标（USD）➤+Add cash➤Digital Dollars➤Tegher USD➤Tron Network➤出现trc20地址
  • 其他入金方式自行尝试

开启美元卡

！！开通美元卡需3美元开卡费，每月1美元的月费！！

  • card➤选择美元卡（看卡片右上角usd）➤点击Claim USD card➤按照提示一直点击下一步即可
  • 奈拉卡同理

转奈拉

！！转换的金额太小会报错，自行翻译查看！！

  • home（主页面）➤点击美国图标（USD）➤Convert➤输入想要转换的金额➤next
  • 奈拉转美元，即点击尼日图标（NGN），然后进行一样的操作

visa卡的转入和转出

  • 加号：转入
  • 斜向下箭头：转出
  • 眼睛：查看卡片信息
  • Statements：账单

chipper cash的转出

- Transfer➤Chipper Tag 或 NGN Bank Accounts

  • chipper tag：chipper账户之间的互转，也就是chhipper好友之间的互转（输入对方用户名即可）
  • NGN Bank Account：尼日利亚银行之间的互转（选择对应银行，输入银行账号，检查用户名即可）
